# JSON Viewer & Formatter — Clean, Private, No Ads

A fast, **privacy-first** JSON viewer and formatter for Chrome. Automatically formats, beautifies, and validates JSON in your browser — collapsible tree view, syntax highlighting, search, dark mode.

**Everything runs 100% locally. No ads. No tracking. No telemetry. No network calls. Open source — audit every line.**

> Built after the most popular JSON extensions turned closed-source, added donation pop-ups, or shipped trackers. This one never will: it's MIT-licensed, requests the **minimum** permissions (`storage` only — no host access), and makes **zero** network requests. Verify it yourself in the source.

## Features
- Auto-formats raw JSON API responses and `.json` pages
- Collapsible / expandable tree + raw toggle
- Syntax highlighting
- Search / filter within large JSON
- One-click copy (pretty-printed)
- Dark mode, works fully offline
- Manifest V3, zero host permissions

## Install
1. **Chrome Web Store:** _(listing pending)_
2. **From source:** clone → `chrome://extensions` → Developer mode → Load unpacked → select this folder.

## Privacy
This extension makes **no network requests of any kind** and collects **nothing**. Your JSON never leaves your browser. The `manifest.json` declares only the `storage` permission and **no** `host_permissions`. That's the whole privacy policy.
