## Detailed description
```
A fast, privacy-first JSON viewer and formatter for Chrome. Automatically formats, beautifies, and validates JSON in your browser — collapsible tree view, syntax highlighting, one-click copy, and search. Everything runs 100% locally: no ads, no tracking, no telemetry, no external servers. Open source and Manifest V3.

Features:
• Auto-format raw JSON API responses and .json pages
• Collapsible / expandable tree + raw toggle
• Syntax highlighting
• Search within large JSON
• One-click copy (pretty-printed)
• Dark mode, works fully offline
• Requests only the "storage" permission — no host access, zero network calls

Tired of JSON extensions that turned closed-source, injected pop-ups, or added tracking? This one never will: it's open source (MIT), requests the minimum permissions, and makes zero network requests — verify it yourself in the source: https://github.com/lordbasilaiassistant-sudo/json-viewer-extension
```
